# rin-re [![Build Status](https://circleci.com/gh/BangumiMoe/rin-re.svg?style=svg)](https://circleci.com/gh/BangumiMoe/rin-re)

Bangumi Moe - refined.

## Development

```shell
$ npm install
$ npm run dev # 127.0.0.1:3000
```

### Build

```shell
$ npm run build
```

### Lint / Format

```shell
$ npm run lint
$ npm run format
```

### Analyze

```shell
$ npm run analyze
```
